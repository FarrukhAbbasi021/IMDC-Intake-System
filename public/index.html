🚀 Setup Guide

Follow these steps to get ViralFlow fully running — takes about 30 minutes

✅ Step 1 — Already Done!

ViralFlow is deployed on Render and running. Your app URL is shown in the Dashboard.

Step 2 — Fix Your API Keys

!

Fix GROQ_API_KEY

Your current key starts with xai- which is an xAI/Grok key, not a Groq key. They are different services.

1. Go to console.groq.com → Sign up free
2. API Keys → Create New Key — it will start with gsk_
3. Add it as GROQ_API_KEY in Render → Environment → Redeploy

B

Pexels API (FREE — Stock Videos & Images)

1. Go to pexels.com/api → Create free account
2. Your API Key is on the API page — copy it
3. Add as PEXELS_API_KEY in Render Environment

Step 3 — Fix Environment Variables in Render

3

Required Variables

Go to your Render service → Environment tab. Make sure all these are set:

APP_URL=https://viralflow-1xsh.onrender.com SESSION_SECRET=any-random-32-char-string NODE_ENV=production ADMIN_USERNAME=admin ADMIN_PASSWORD=your-password-here GROQ_API_KEY=gsk_xxxx ← must start with gsk_ PEXELS_API_KEY=your-pexels-key YOUTUBE_CLIENT_ID=your-client-id YOUTUBE_CLIENT_SECRET=your-client-secret YOUTUBE_REDIRECT_URI=https://viralflow-1xsh.onrender.com/auth/youtube/callback

⚠️ After adding/changing any variable in Render, click Manual Deploy → Deploy latest commit to apply changes.

Step 5 — Enable Auto-Pilot

1️⃣
Add a Niche

Go to Niches → Add topics for AI to create content about (e.g. "AI Tools", "Make Money Online")

2️⃣
Turn on Auto-Generate

Settings → Auto-Generate Daily Content → ON. AI will generate a new video every day at 6 AM.

3️⃣
Turn on Auto-Publish

Settings → Auto-Publish at Optimal Time → ON. Posts will go live at peak engagement times automatically.

4️⃣
Test it First

Generate a video manually → review the script → click Publish to test the full workflow before enabling auto-publish.
